COMANDOS HTML 5
Comando Función
(html - /html) Nos abrirá y cerrará el cuerpo del html
(head - /head) Este comando será para el encabezado de nuestra web
(body - /body) Aquí podremos modificar el cuerpo de nuestra web
(tittle - /tittle) Nombre de nuestra web en la pestaña
(link) Nos permite linquear algun enlace externo
(h1-h2-h3-h4-h5-h6) Comando que nos permitirá introducir texto con diferentes tamaños
(table - /table) Introduce una tabla
(thead - /thead) Será el encabezado de nuestra tabla
(tbody - /tbody) Este será para el cuerpo de la tabla
(tr - /tr) Añadirá una nueva fila en la tabla
(th - /th) Añadirá una nueva columna en la tabla
(p - /p) Es usado para introducir párrafo
(a - /a) Nos permitirá introducir hiperenlace
(strong - /strong) Introduce un texto con importancia
(n - /n) Nos dará un texto en negrilla
(br - /br) Hará un espacio en nuestra web, separando elementos
(hr - /hr) Hará un espacio en nuestra web, separando elementos mediante un vector horizontal
(img) Inserta contenido multimedio en nuestra web
(center - /center) Ajusta en el centro de la web los elementos dentro de este comando
border Nos dará el borde de algún elemento
(height - width) Moficia ancho y alto del elemento
(ol - /ol) Nos hará una lista en orden numerico
(li - /li) Hará una lista
(div - /div) Permite separar uno o varios elementos
(button - /button) Añade un botón para hiperenlaces
(class="") Separa un elemento bajo una etiqueta en especifico
